# CareSync HMS (Frontend)

CareSync HMS is a **frontend-only Hospital Management System interface** built using **HTML, CSS, and JavaScript**.  
The project focuses on creating a clean, responsive, and user-friendly UI for hospital workflows such as patient management, appointments, billing, and dashboards.

This version does **not include any backend or database integration** and is intended for UI demonstration, prototyping, or academic/hackathon purposes.
